# Cross-Paper Findings

This folder contains synthesis notes linking findings across papers. Each file covers a concept or hypothesis and tracks how well it is supported across studies.

## How to use

- One file per concept/hypothesis (e.g., `clonality-as-driver-timing.md`)
- Each file lists supporting and contradicting evidence from `papers/`
- Populated as more papers are added
